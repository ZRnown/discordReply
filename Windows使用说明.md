# Discord自动回复工具 - Windows使用说明

## 📦 安装说明

1. **下载文件**
   - 从GitHub Actions构建产物中下载 `DiscordAutoReply.exe`
   - 文件大小约50MB，包含所有必要组件

2. **放置文件**
   - 将 `DiscordAutoReply.exe` 放在您选择的文件夹中
   - 建议创建专用文件夹，如 `C:\DiscordAutoReply\`

3. **运行程序**
   - 双击 `DiscordAutoReply.exe` 启动程序
   - 如果出现安全警告，点击"更多信息" → "仍要运行"

## 🚀 基本使用

### 1. 获取Discord Token

**重要**：Token是您的Discord登录凭证，请妥善保管，不要泄露给他人。

#### 方法1：浏览器获取（推荐）
1. 打开 [Discord网页版](https://discord.com/app)
2. 按 `F12` 打开开发者工具
3. 切换到 `Application` 标签页
4. 左侧选择 `Local Storage` → `discord.com`
5. 找到 `token` 字段，复制 `value` 列的值

#### 方法2：Discord客户端
1. 打开Discord桌面客户端
2. 按 `Ctrl+Shift+I` 打开开发者工具
3. 按照上述步骤操作

### 2. 配置程序

1. **启动程序**：双击运行 `DiscordAutoReply.exe`

2. **添加账号**：
   - 点击"添加账号"按钮
   - 粘贴您获取的Discord Token
   - 点击"验证Token"确认有效性
   - 保存账号

3. **配置规则**：
   - 点击"添加规则"按钮
   - 设置关键词（如：`hello`）
   - 设置回复内容（如：`你好！`）
   - 选择匹配类型（部分匹配/精确匹配/正则表达式）
   - 设置延迟时间（避免被检测）
   - 保存规则

4. **账号规则关联**（可选）：
   - 在账号列表中点击账号的"规则"按钮
   - 选择该账号要应用的规则
   - 未选择的规则不会对此账号生效

### 3. 启动自动回复

1. **点击"启动"按钮**
2. **查看运行日志**：确认连接成功
3. **测试功能**：
   - 在Discord中发送包含关键词的消息
   - 机器人会自动回复

## ⚙️ 高级配置

### 规则匹配类型

- **部分匹配**：消息包含关键词即触发（如关键词"hello"，消息"hello world"会触发）
- **精确匹配**：消息完全等于关键词（如关键词"hello"，只有消息"hello"会触发）
- **正则表达式**：使用正则表达式匹配，支持复杂模式

### 延迟设置

- **最小延迟**：回复前最少等待时间
- **最大延迟**：回复前最多等待时间
- **实际延迟**：程序随机选择介于最小和最大之间的值
- **建议设置**：2-5秒，避免被Discord检测

### 频道限制

- **全部频道**：在所有加入的服务器频道中监听
- **指定频道**：只在特定频道ID中监听
- **如何获取频道ID**：
  1. Discord设置 → 高级 → 启用开发者模式
  2. 右键频道 → 复制ID

## 🔧 故障排除

### 程序无法启动
- 确保您的Windows版本支持（Windows 10+）
- 尝试以管理员身份运行
- 检查杀毒软件是否拦截

### Token验证失败
- 检查Token是否正确复制
- 确认Token格式（通常以字母开头）
- 尝试重新获取Token

### 无法自动回复
- 确认机器人已启动（状态显示"运行中"）
- 检查规则配置是否正确
- 确认账号有频道发送权限
- 查看运行日志了解详细错误

### 连接错误
- 检查网络连接
- 等待几分钟后重试
- 如果频繁出现，可能是网络或Discord服务器问题
